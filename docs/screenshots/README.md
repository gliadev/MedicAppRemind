## Cómo generarlas (simulador)

1. Ejecuta en el simulador del dispositivo de referencia (ES y EN).
2. Pobla datos de ejemplo coherentes (3–4 medicamentos, uno en stock bajo).
3. Para la 05: Ajustes ▸ Accesibilidad ▸ Pantalla y tamaño de texto ▸ tamaños AX.
4. Captura con `Cmd+S` en el simulador y renombra según la tabla.

> Plan completo (tamaños App Store 6.9"/6.5", set ES/EN): `../appstore/Screenshots.md`.
